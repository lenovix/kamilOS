1️⃣ Boot & Kernel Initialization

Fitur dasar sistem operasi yang memuat kernel ke memori dan memulai eksekusi setelah bootloader.

File Terkait

  • boot.asm — Bootloader, memuat kernel.
  • kernel.c — Entry point utama.

Contoh Alur

Boot → Load kernel → Initialize screen → Show logs

2️⃣ Screen & Text Handling

Modul ini menangani tampilan teks ke layar menggunakan akses langsung ke memori video (VGA text mode).

File Terkait

  • src/drivers/screen.c
  • src/include/screen.h

Fungsi Utama

  • void put_char(char c, uint8_t color) — Menulis satu karakter di posisi kursor.
  • void print_string(const char *str, uint8_t color) — Menampilkan teks berwarna.
  • void print_line(const char *prefix, const char *msg, uint8_t color) — Menampilkan teks dengan awalan (seperti log).
  • void clear_screen() — Menghapus seluruh tampilan layar.

Contoh Penggunaan

print_string("Hello, kamilOS!", WHITE_ON_BLACK);
print_line("[SYSTEM]", "Screen initialized", GREEN_ON_BLACK);
clear_screen();

3️⃣ Basic Logging

Sistem logging sederhana untuk menampilkan pesan dengan level tertentu (INFO, WARN, ERROR, DEBUG) di layar.

File Terkait

  • src/kernel/log.c
  • src/include/log.h

Fungsi Utama

  • void log_info(const char *msg) — Menampilkan pesan informasi.
  • void log_warn(const char *msg) — Menampilkan pesan peringatan.
  • void log_error(const char *msg) — Menampilkan pesan error.
  • void log_debug(const char *msg) — Menampilkan pesan debug.

Contoh Penggunaan

log_info("Kernel started");
log_warn("Low memory");
log_error("Disk not found");
log_debug("Entering test mode");

Output di Layar

[INFO] Kernel started
[WARN] Low memory
[ERROR] Disk not found
[DEBUG] Entering test mode
